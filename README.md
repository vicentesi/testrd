## Live Demo

[TestRD](https://testrd.herokuapp.com/)

## Installation

...
bundle install

Run server: (default: [localhost:3000](http://localhost:3000))
```rails server```

## Tests

* RSpec
* Capybara
* Spork
* Guard

Running tests:
```bundle exec rspec spec/[filename]```
or:
```time rspec spec/[filename]```

To run guard with spork server, run:
```bundle exec guard```
